# PIXS Printing Shop

Premium Printing Solutions for businesses and individuals. This repository contains the integrated ERP and E-commerce platform for PIXS Printing Shop, featuring a modern React frontend and a robust Laravel backend with PostgreSQL.

## 🚀 Technology Stack

### Frontend

- **Framework:** React + TypeScript
- **Styling:** Vanilla CSS / Tailwind (Modern, industrial design)
- **State Management:** Store-based architecture
- **Animations:** Motion-driven UI (GSAP)

### Backend

- **Framework:** Laravel 11
- **Database:** PostgreSQL
- **Real-time:** Integrated communication hub for staff and customers
- **Security:** 1

## 🛠️ Features

- **Order Tracking:** Robust order lifecycle management from pending to delivery.
- **Inventory System:** Real-time stock alerts and automated restock logs.
- **Payroll & Attendance:** Enterprise-grade engine for employee management.
- **Customer Portal:** Seamless product discovery and order placement.
- **Screenplate Management:** Specialized tracking for custom printing plates.

## 📖 Project Structure

- `/frontend`: React client application.
- `/backend`: Laravel API and database management.
- `/frontend/src/data`: Mock JSON data used for initial development (currently being migrated to PostgreSQL).

## 🏁 Getting Started

### Development

1. **Frontend:**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
2. **Backend:**
   ```bash
   cd backend
   composer install
   npm install
   composer run dev
   ```
